# Recruitment Brief — One Paragraph

We are building a web-based algorithmic options trading platform for NIFTY and SENSEX, hosted on AWS, with a Python (FastAPI) backend and React frontend, integrated with Axis Direct's RAPID API. The platform will be operated by non-technical traders via a clean UI with dropdowns and inputs — no code changes needed for daily use. Phase 1 delivers a manual-strike, combined-premium-trigger execution engine with full risk controls (hard SL, time-based exit, position cap, kill switch, circuit breaker, audit logging). Phase 2 adds intelligent strike selection (distance-based, percentage-based, delta-based with Black-Scholes Greeks). Phase 3 is a separate market intelligence dashboard (live OI, max pain, India VIX, FII/DII flows, news). We will write the initial code ourselves using Claude Code, and we are looking for an experienced agency or senior freelancer to do a comprehensive production-readiness audit, harden the code (error handling, state reconciliation, idempotency, testing framework), set up AWS infrastructure properly (monitoring, alerting, secrets management, deployment pipeline), supervise paper trading validation across 4-6 expiries, and provide 60-90 day post-launch support. Required experience: live deployment of trading bots with Indian broker APIs (Zerodha Kite, Angel SmartAPI, Fyers, or similar), Python async expertise, AWS production deployments, understanding of SEBI algo trading compliance. Engagement is fixed-cost or milestone-based, not hourly. Please share: relevant case studies, your approach to order state reconciliation, testing methodology, IP ownership terms, and post-go-live SLA.
